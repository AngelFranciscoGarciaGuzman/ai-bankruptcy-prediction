# Taiwanese Bankruptcy Prediction

**Autor:** Angel Francisco Garcia Guzman — A01704203

## Contexto

Este proyecto construye un modelo de clasificación binaria para predecir si una empresa entrará en bancarrota a partir de sus indicadores financieros.

**Dataset:** *Taiwanese Bankruptcy Prediction* — recopilado del *Taiwan Economic Journal* (1999–2009). La definición de bancarrota se basa en las regulaciones de la *Taiwan Stock Exchange*.

| Característica | Valor |
|---|---|
| Tipo de problema | Clasificación binaria |
| Área | Negocios / Finanzas |
| Instancias | 6,819 empresas |
| Features | 95 indicadores financieros |
| Tipo de feature | Numérico (ya normalizado a [0,1]) |
| Variable objetivo | `Bankrupt?` (0 = no, 1 = sí) |
